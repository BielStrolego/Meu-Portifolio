Sobre mim

Olá! Meu nome é Gabriel Strolego Queiroz e nasci em agosto de 2000.
Meu caminho incluiu a fundação da empresa G-Logic e formação técnica em informática.
Além de fornecer suporte técnico para empresas e serviços de gráfica, a G-Logic oferece manutenção de computadores, notebooks e celulares, criação de sistemas, sites e aplicativos, instalação de câmeras de segurança e infraestrutura.
Como desenvolvedor Full Stack, estou sempre entusiasmado em usar a tecnologia para fazer ideias acontecerem.

Angular

Angular é um framework de código aberto desenvolvido pelo Google para a criação de aplicativos dinâmicos e interativos da web.
O Angular oferece uma estrutura sólida para o desenvolvimento, incluindo suporte a gerenciamento de estado, roteamento, validação de formulários e muito mais.

HTML5

O HTML5 é a quinta versão da linguagem de marcação padrão para páginas web.
O HTML (HyperText Markup Language) é o código usado para estruturar uma página web e seu conteúdo. Por exemplo, ele permite estruturar o conteúdo em parágrafos, listas com marcadores, imagens e tabelas.

CSS3

O CSS3 é a terceira versão das Cascading Style Sheets, também conhecidas como CSS.
Essa linguagem é usada para definir estilos em projetos web, como páginas da internet. Com o CSS3, é possível criar estilos únicos para seus projetos, incluindo efeitos de transição, imagens de fundo e outros aspectos de design no layout da página.

JavaScript

JavaScript é uma linguagem de programação de uso geral, aplicada principalmente para desenvolvimento web e desenvolvimento de software.
Especificamente, o JavaScript atua na programação front-end, que é a parte “visual” de uma aplicação, geralmente um site ou um aplicativo.
Ele permite criar conteúdo que se atualiza dinamicamente, controlar multimídias, imagens animadas e muito mais.

TypeScript

O TypeScript é uma linguagem de programação de código aberto desenvolvida pela Microsoft. Ele é um superconjunto sintático estrito do JavaScript, o que significa que adiciona tipagem estática opcional à linguagem.
Em termos simples, o TypeScript oferece uma camada adicional de segurança e clareza ao seu código, garantindo que erros sejam detectados antes mesmo de você executar o programa.

Bootstrap

O Bootstrap é um framework front-end de código aberto usado para desenvolver sites e aplicações web.
Ele combina CSS e JavaScript para estilizar elementos de páginas escritas em HTML.
O Bootstrap é amplamente conhecido por sua capacidade de criar layouts responsivos que se adaptam a diferentes tamanhos de tela, incluindo dispositivos móveis.

Ionic

O Ionic é um framework de código aberto que facilita a construção de aplicativos móveis híbridos. Ele utiliza tecnologias web como HTML, CSS e JavaScript para criar soluções que podem ser executadas em várias plataformas, como iOS, Android e Windows.
Desenvolvido por Max Lynch, Ben Sperry e Adam Bradley da Drifty Co., o Ionic ajuda as equipes a criarem e distribuírem belos aplicativos híbridos, com foco na experiência do usuário e interação com o aplicativo.

Node.js

O Node.js é um ambiente de execução JavaScript que permite rodar aplicações desenvolvidas com essa linguagem do lado do servidor.
Diferentemente dos navegadores, o Node.js possibilita criar diversos tipos de aplicações web, desde servidores para sites estáticos e dinâmicos até APIs e sistemas baseados em microserviços.

Java

Java é uma linguagem de programação orientada a objetos criada na década de 90 pela equipe liderada por James Gosling na Sun Microsystems, que posteriormente foi adquirida pela Oracle Corporation.
Ela se tornou amplamente popular devido à sua capacidade de escrever código uma vez e executá-lo em diferentes dispositivos, desde sites e computadores até datacenters, celulares e videogames.

Python

Python é uma linguagem de programação de alto nível, o que significa que sua sintaxe é mais simplificada e próxima da linguagem humana.
Ela é amplamente utilizada em diversas aplicações, como desktop, web, servidores e ciência de dados
Algumas vantagens que o tornam uma escolha popular são: Simplicidade, Implantação Rápida e Alto Desempenho.

PHP

O PHP é uma linguagem de programação de script utilizada na criação de sites e sistemas para a web.
Ela é interpretada e executada no lado do servidor, permitindo que os desenvolvedores criem páginas dinâmicas que interajam com os usuários e outros sistemas.

Mysql

Basicamente, o MySQL permite que você armazene, recupere e manipule informações em tabelas organizadas de forma relacional.
Ele é amplamente utilizado por aplicativos e sites populares, como Facebook, Twitter, YouTube e Google.
